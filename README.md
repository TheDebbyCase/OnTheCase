# On-The Case

A cosmetics library that should allow modders to be able to add their own cosmetics!

You can use the [Issues](https://github.com/TheDebbyCase/OnTheCase/issues) tab to submit any problems with the mod, or any suggestions you may have.
Suggestions can also be made directly to me if you can find me on Discord, or on [Bluesky](https://bsky.app/profile/thedebbycase.bsky.social)!

If you wish you can also donate to me over on [Ko-Fi](https://ko-fi.com/thedebbycase)!

See the Thunderstore page [here](https://thunderstore.io/c/on-together/p/deB/OnTheCase),
or the GitHub page [here](https://github.com/TheDebbyCase/OnTheCase)!
